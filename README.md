📊 Sales Prediction: Multi-Channel Advertising AnalysisOptimizing Marketing ROI through Multiple Linear Regression and Statistical Modeling📝 Project OverviewIn the modern business landscape, data-driven decision-making is the difference between a successful campaign and a wasted budget. This project addresses the Marketing Mix Modeling (MMM) problem: identifying how much "Sales" ($y$) are generated by spending on different "Advertising Channels" ($X$).By utilizing Multiple Linear Regression, this project creates a mathematical framework to predict future sales performance with 89.94% accuracy. The ultimate goal is to provide a "Decision Support System" that tells a business exactly where to invest their next marketing dollar for maximum impact.📖 Dataset FeaturesThe dataset consists of 200 observations, representing the sales of a product in different markets along with the advertising budgets allocated to three media channels.FeatureTypeDescriptionTVNumericalAdvertising budget spent on TV ads (in thousands of dollars).RadioNumericalAdvertising budget spent on Radio ads (in thousands of dollars).NewspaperNumericalAdvertising budget spent on Newspaper ads (in thousands of dollars).SalesTargetThe number of units sold (in thousands). This is the value we aim to predict.⚙️ Project WorkflowFollowing the CRISP-DM (Cross-Industry Standard Process for Data Mining) methodology, the project was executed in five distinct phases:Data Integrity Validation: Conducted checks for null values and duplicate rows to ensure the "Golden Source" of data was clean.Exploratory Data Analysis (EDA): Performed statistical summaries and correlation analysis to find "Hidden Patterns."Data Preprocessing: Handled outliers in the Newspaper feature and split the data into Training (80%) and Testing (20%) sets.Model Implementation: Trained a Multiple Linear Regression algorithm using the Scikit-Learn library.Performance Evaluation: Validated the model using $R^2$ Score, MAE, and RMSE to ensure high predictive reliability.🎨 Visual Analysis & InsightsVisualizations were key to understanding the "Business Logic" of the data:Correlation Heatmap: Revealed that TV and Radio have a strong positive relationship with Sales, while Newspaper shows a very weak link.Boxplots: Used to identify outliers. We found two instances of exceptionally high Newspaper spend that did not result in higher sales.Actual vs. Predicted Plot: A scatter plot with a "Line of Best Fit" shows our model's guesses are extremely close to the real-world results.Residual Analysis: Confirmed that the errors are randomly distributed, proving the Linearity of our data.Shutterstock Explore 🛠️ Technologies UsedLanguage: Python 3.xData Manipulation: Pandas, NumPyData Visualization: Matplotlib, SeabornMachine Learning: Scikit-Learn (LinearRegression, train_test_split, metrics)Environment: Jupyter Notebook / Kaggle🚀 How to UseTo run this project on your local machine:Clone the Repository:Bashgit clone https://github.com/YourUsername/Advertising-Sales-Prediction.git
