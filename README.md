# 📊 Sales Prediction: Multi-Channel Advertising Analysis

## 📝 Project Overview
This project addresses the **Marketing Mix Modeling (MMM)** problem: identifying how much **Sales** are generated by spending on different advertising channels.

By utilizing **Multiple Linear Regression**, this project predicts future sales performance with **89.94% accuracy**.

---

## 📖 Dataset Features
- **TV**: Budget spent on TV ads (in thousands of dollars)
- **Radio**: Budget spent on Radio ads (in thousands of dollars)
- **Newspaper**: Budget spent on Newspaper ads (in thousands of dollars)
- **Sales (Target)**: Number of units sold (in thousands)

---

## ⚙️ Project Workflow

### ✅ Data Integrity
- Verified zero missing values and zero duplicate rows

### 📊 Exploratory Data Analysis (EDA)
- Used correlation heatmaps
- Identified that **TV and Radio** are the primary sales drivers

### 🧹 Preprocessing
- Handled outliers in Newspaper spend
- Split data into **80% Training / 20% Testing**

### 🤖 Modeling
- Trained a **Multiple Linear Regression** model

### 📈 Evaluation
- Evaluated using **R-Squared Score, MAE, and Residual Plots**

---

## 🎨 Visual Analysis & Insights
- **Heatmap**: Confirmed TV and Radio have strong positive correlation with Sales  
- **Boxplots**: Showed that high Newspaper spend does not significantly increase Sales  
- **Residual Plot**: Confirmed errors are random, indicating an unbiased model  

---

## 🛠️ Technologies Used
- **Python** (Pandas, NumPy)  
- **Scikit-Learn** (Modeling & Metrics)  
- **Matplotlib & Seaborn** (Visualization)  

---

## 💡 Final Strategic Recommendation
Reallocate the **Newspaper** budget to **Radio**.  
The model shows that Newspaper advertising has near-zero impact on sales, while Radio offers the highest return per dollar spent.  

Shifting these funds can increase total sales **without increasing the overall marketing budget**.
